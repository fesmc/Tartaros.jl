# Tartaros

Geothermal heat flux (GHF) is a key boundary condition for ice sheet models, but is often poorly constrained by observations. Tartaros.jl provides a flexible framework that includes:
 - Simple thermodynamic models to compute the interaction between the temperature at the ice base and the GHF,
 - Correction methods that can improve reference GHF measurements.
